# vibeOS
Que es VibeOS
VibeOS es un entorno de escritorio web que integra un conjunto de herramientas legales potenciadas por IA en una sola interfaz tipo OS. Funciona como un archivo HTML autocontenido: sin backend propio, sin build step, sin dependencias npm. Solo abre el archivo en el navegador y opera.
Casos de uso:
Firmas legales que necesitan un asistente IA sobre normativa local
Empresas que deben cumplir con leyes de datos (Ley 21.719, GDPR, LGPD, etc.)
Desarrolladores que quieren un punto de partida para legal-tech con IA
Demos y MVPs de productos legales con IA
Caracteristicas
Modulo
Descripcion
Agente Legal IA
Chat con cualquier LLM (Claude, GPT, Mistral, Groq, Ollama) + contexto de normativa activa
Normativa Manager
Carga, edita y activa normativas de cualquier pais con articulos especificos
WhatsApp Gateway
Integracion con Twilio y Meta Cloud API para recibir y responder mensajes
Anonimizador PII
Deteccion y mascara de datos personales (RUT, DNI, CPF, IBAN, email, telefono, etc.)
Compliance Wizard
Diagnostico interactivo de cumplimiento Ley 21.719 en 5 pasos
Terminal
Consola de comandos para controlar el sistema desde teclado
Configuracion
Panel centralizado para API keys, webhooks, comportamiento del agente
